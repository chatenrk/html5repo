# Emmet Usage - Cheat Sheet
```sh
#header
<div id="header"></div>
```
```sh
.title
<div class="title"></div>
```
```sh
form#search.wide
<form id="search" class="wide"></form>
```
```sh
p.class1.class2.class3
<p class="class1 class2 class3"></p>
```
